1. Create `src/knowledge/review-comment-embedding-sweep.ts` with the `sweepNullEmbeddings` function.

2. The function flow:
   a. Get initial count via `store.countNullEmbeddings(repo)`. Log start with `{ repo, totalNull }`.
   b. If totalNull === 0, log and return early with zeroed result.
   c. Loop: fetch batch via `store.getNullEmbeddingChunks(repo, batchSize)`.
   d. If batch is empty, break.
   e. For each chunk in batch:
      - Try `embeddingProvider.generate(chunk.chunkText, "document")`
      - If result is non-null and not dryRun: call `store.updateEmbedding(chunk.id, result, "voyage-code-3")`
      - Increment succeeded/failed counters

   f. After each batch: log progress `{ repo, batchNumber, processed, succeeded, failed }`.
   g. Sleep `batchDelayMs` between batches (skip sleep after last batch).
   h. Respect `maxBatches` -- if set, break after that many batches.
   i. Log completion with `{ repo, totalNull, processed, succeeded, failed, durationMs }`.
   j. Return `EmbeddingSweepResult`.

3. Use a simple `sleep` helper (can import from backfill or define locally -- define locally to avoid coupling).

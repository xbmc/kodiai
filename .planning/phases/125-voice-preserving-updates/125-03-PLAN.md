       - `buildVoicePreservingPrompt(opts: { styleDescription: PageStyleDescription; exemplarSections: StyleExemplar[]; originalSection: string; sectionHeading: string | null; diffEvidence: string; }): string`
         - Build a generation prompt that:
           a. Starts with system context: "You are updating a wiki page section. Your output MUST match the existing page's voice, tone, and formatting."
           b. Includes the style description under "## Page Style"
           c. Includes exemplar sections under "## Style Examples (from this page)" — each with heading and content

              - "Do NOT add, remove, or reorder sections — update content within the existing section only"

           g. Returns the full prompt string

           b. Calls extractPageStyle once for the page
           c. Calls selectExemplarSections once for the page
           d. For each section in the input array:

                - originalSection: section.chunkText

                - buildPromptWithFeedback: (feedback) => rebuild prompt with feedback appended
              - Collects results into VoicePreservedUpdate array
           e. Logs summary: `{ pageId, sectionsProcessed, voiceMismatches }` at info level
           f. Returns VoicePreservedUpdate array
